# MOTIPE - 지역 축제 기반 AI 여행 추천 플랫폼

## 프로젝트 소개

사용자의 성향을 분석해 최적화된 지역 축제 및 여행 일정을 추천하는 AI 플랫폼입니다.

## 기술 스택

- Frontend: HTML5, CSS3, JavaScript ES6+
- Backend: Node.js, Express.js
- Database: Supabase

## 주요 기능

- 사용자 축제 성향 진단 및 맞춤형 칭호 부여
- AI 기반 축제 연계 여행 일정 생성 (최대 3개 키워드)
- 커뮤니티 게시판 (리뷰 및 꿀팁 공유)

## AI 추천 알고리즘

추천 점수 = 축제 테마 적합도 50% + 일정&날씨 적합도 20% + 동선 15% + 동행자 15%

## 실행 방법

```bash
npm install
npm start
```
